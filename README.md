Write-good on Glitch
====================

Uses [write-good](https://github.com/btford/write-good) to provide suggestions for English prose.

The Project
-----------

On the back-end,
- the app starts at `server.js`
- frameworks and packages live in `package.json`
- app secrets ALWAYS go in `.env`

On the front-end,
- edit `client.js`, `style.css` and `index.html`
- drag in `assets`, like images or music, to add them to the project


About Glitch
------------

Click `Show` in the header to see your app live. Updates to your code will instantly deploy and update live.

Glitch is a developer playground that lets you code a real web-app without the slow setup and deployment steps.
